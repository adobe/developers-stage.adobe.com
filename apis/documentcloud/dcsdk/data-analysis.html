Data Analysis

Access to data is critical to success, but unlocking historical information is difficult and time consuming. Important, older data is often inaccessible because it is still on paper, or, if it has been scanned, it is not easy to find. With digital archives virtually overflowing with thousands of documents, including PDFs, organizations want to accelerate the process of making this information readily available.

Using Adobe Document Services, companies can:

  • Export PDF documents into other formats, such as Microsoft Excel
  • Perform analysis on the data in these documents
  • Use the processed data to create new reports or whitepapers

Relevant APIs

  • Adobe PDF Tools API
  • Adobe PDF Embed API

Scenario

A company has several years of archived financial reports that have been saved in different formats. Some of the files are also paper documents, so data is difficult to use without significant manual intervention. Data analysts want to be able to select any reports from a repository, no matter its original format, and analyze and perform calculations on them. After completing their analysis, they want to create new reports to share with executives in PDF format.

Building this Solution

  1. Scan paper documents to PDF format to create a digital repository
  2. Create an interface that presents select files for a data analyst to choose from
  3. Export selected PDF documents to XLS using Adobe PDF Tools API export operation
  4. Perform calculations on multiple XLS files for new insights and store in a database
  5. Using Adobe PDF Tools API, recreate charts and data in PDF format as a report or whitepaper
  6. Display new documents within the application with Adobe PDF Embed API
